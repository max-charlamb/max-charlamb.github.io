Built a compiler for toy language xi as part of CS 4120/4121. Xi language spec can be found here, the type spec can be found here, and the ABI spec (application binary interface) can be found here.
CUAir

Cornell University Unmanned Air Systems is a student run project team which builds a autonomous drone to compete in the AUVSI SUAS competition. The regularly competes at the highest levels.

I worked on both software and electrical systems. During my time I focused on the antenna tracker, creating an electrical PCB using Altium and AVR microcontrollers which tracked the planes position in real time and ran motor controllers to adjust the angle of the ground antenna.
